Privacy Policy

Last Updated: March 21, 2026

At PULLEDOUT.LOL ("we", "us", or "our"), we take your privacy seriously. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our Advanced Web Security Analysis Platform ("Service").

📌 Quick Summary

We use Discord OAuth for authentication and require server membership for access. We collect minimal data necessary to operate the Service. Payment is processed securely through LemonSqueezy. We do not sell your information.

1. Information We Collect

1.1 Discord Account Information

When you authenticate using Discord OAuth, we collect:

  • Discord User ID: Your unique Discord identifier
  • Discord Username: Your Discord username and discriminator (if applicable)
  • Discord Avatar: Your Discord profile avatar URL
  • Server Membership: Verification that you are a member of our designated Discord server
  • Authentication Tokens: Temporary OAuth tokens (not stored permanently)

Note: We do NOT store your Discord password. Authentication is handled securely through Discord's OAuth 2.0 system.

1.2 Payment Information

Payment processing is handled by LemonSqueezy. We collect:

  • Transaction ID: Unique identifier for your purchase
  • Payment Status: Success/failure status of transactions
  • Purchase Date: Timestamp of your access purchase
  • Payment Amount: Amount paid for access

Important: We do NOT store credit card numbers, CVV codes, or banking details. All payment card data is processed and stored securely by LemonSqueezy in compliance with PCI-DSS standards.

1.3 Scan Data

When you use our security scanning features, we collect and store:

  • Target URLs: Website addresses and domains you submit for scanning
  • Scan Results: Vulnerability findings, security issues, and analysis data
  • Scan Metadata: Date, time, scan settings (max pages), and scan duration
  • Generated Reports: HTML reports generated from scans
  • Scan History: Record of all scans performed under your account (limited retention)

1.4 Technical Information

We automatically collect certain technical data when you use the Service:

  • IP Address: Your internet protocol address for security and fraud prevention
  • Browser Information: Browser type, version, and language preferences
  • Device Information: Operating system and device type
  • Usage Data: Pages visited, features used, timestamps, and interaction patterns
  • Performance Data: Load times, errors, and service performance metrics
  • Session Tokens: Secure authentication session tokens (7-day expiration)

1.5 Discord Bot Data

Our Discord bot collects:

  • Server Join Events: When you join the Discord server via invite link
  • Server Leave Events: When you leave or are removed from the server
  • Invite Link Usage: Tracking of temporary invite links for security

1.6 Information We Do NOT Collect

  • Real names or personal identification documents
  • Email addresses (unless provided by Discord OAuth scope)
  • Credit card or banking information (handled by LemonSqueezy)
  • Passwords (Discord OAuth handles authentication)
  • Sensitive personal data (health, biometric, or financial data)
  • Social media content beyond Discord profile
  • Precise geolocation (only IP-based country/region)
  • Discord messages or private communications

4. Data Security

We implement industry-standard security measures to protect your information:

4.1 Technical Safeguards

  • Encryption: HTTPS/TLS encryption for all data in transit
  • Session Security: Secure 7-day session tokens with HTTPOnly and SameSite flags
  • Database Security: Encrypted PostgreSQL connections and access controls
  • OAuth Security: Discord OAuth 2.0 with state validation
  • CSRF Protection: Cross-Site Request Forgery protection on all forms
  • Rate Limiting: API rate limits to prevent abuse

4.2 Payment Security

  • All payment processing handled by PCI-DSS compliant LemonSqueezy
  • We never store credit card or banking information
  • Transaction data encrypted in transit and at rest
  • Webhook signatures verified for payment confirmations

4.3 Organizational Safeguards

  • Limited access to personal data on a need-to-know basis
  • Regular security audits and vulnerability assessments
  • Incident response procedures for security breaches
  • Security monitoring and logging

4.4 Security Disclaimer

While we implement strong security measures, no system is 100% secure. You are responsible for:

  • Maintaining the security of your Discord account
  • Using strong passwords for Discord
  • Enabling two-factor authentication on Discord (recommended)
  • Not sharing your account or server access with others
  • Reporting any unauthorized access immediately

5. Data Retention

We retain your information for as long as necessary to provide the Service and comply with legal obligations:

5.1 Active Accounts

  • Discord Account Data: Retained while you remain in the Discord server
  • Scan Results: Stored for 90 days or until you delete them
  • Usage Logs: Retained for 90 days for security and analytics
  • Session Tokens: 7-day expiration, automatically deleted after

5.2 Payment Records

  • Transaction records retained for 7 years for tax and legal compliance
  • Payment disputes and refunds retained for 2 years
  • Financial data stored securely in compliance with applicable regulations

5.3 Deleted Accounts / Server Departure

  • When you leave the Discord server, your access is immediately revoked
  • Account data is anonymized within 30 days of server departure
  • Scan results are retained for 90 days then permanently deleted
  • Payment records retained for legal compliance (see above)
  • Some data may be retained in encrypted backups for up to 90 days
